# 进度条

> 用途：各组/各章开始和结束时展示阅读进度 | 被引用：mode-standard-chapter, mode-grouped-epic, mode-anthology, mode-library

## 通用规则

- 进度条使用 █ 和 ░ 字符（20 格）
- **按字数加权**，不按章节数均分
- 百分比向下取整
- 已完成格数 = floor(已完成字数 / 总字数 × 20)
- 已完成字数 = 已完成章节的 chapters[].length 之和
- 总字数 = progress.json 中所有 chapters[].length 之和

## 计算示例

以 3 部、总字数 341,136 为例：

| 部 | 字数 | 读完后累计 | 进度 | 格数 |
|----|------|-----------|------|------|
| 贡布雷 | 143,337 | 143,337 | 42% | 8 |
| 斯万的爱情 | 153,521 | 296,858 | 87% | 17 |
| 地名 | 44,278 | 341,136 | 100% | 20 |

## 各模式格式

### Standard Chapter Mode

```
[{第N部名} {████████░░░░░░░░░░░░}] {N}/{total} 部
```

### Grouped Epic Mode

```
[{第N部名} {████████░░░░░░░░░░░░}] {N}/{total} 部
```

### Anthology Mode

```
[{篇名} {N}/{total} 篇] — {合集名}·{辑名}
```

### Library Mode

```
[{合集名} {N}/{total} 本] | {书名}（{类型}·{部数}部）→ 开始
```
