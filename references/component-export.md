# 导出阅读笔记

> 用途：全书/全合集总结后将阅读记录打包导出为 Markdown | 被引用：Final Summary

## 导出路径

`output/{book_slug}/reading-journal.md`

## 内容结构

```markdown
# {书名} · 阅读笔记
> 作者：{作者}
> 难度：{★级} · 类型：{类型}
> 阅读周期：{起止日期}

## 阅读进度
{进度条和完成情况}

## 各章/各部回顾
{逐章/逐部，每章含以下字段：}

### 第X部/第X章：{标题}
- 情节锚点：{一句话}
- 叙事技巧聚焦：{本章最突出的叙事手法}
- 关键意象：{本章引入/变奏的核心意象}
- 引导问题：{L1/L2/L3} — {问题文本}
- 你的回答：{用户回答}
- AI参考：{证据 + 分析 + 延伸}
- 读后一问：{回顾问题}
- 测验：{题目 + 用户答案/正确答案}

## 人物长廊
{初始人物快照 + 阅读过程中新增的人物}

## 全书总结
{final-summary.md 的完整内容}

## 难度回顾
{用户的难度评价}

## 延伸推荐
{个性化推荐或图书馆分类推荐}

> 此笔记由 novel-reader v2.2 自动生成
```

## Library 模式导出

- 每读完成一本书 → 导出该书的 `reading-journal.md`
- 合集总结时 → 导出合集总览 `journal-index.md`，包含所有书的链接和进度总览

## 文件命名

确保导出文件名不冲突（如已存在则追加数字后缀，如 `reading-journal-2.md`）。导出后告知用户路径。
