# 洞察提炼

> 用途：全书完成后，将用户阅读过程中的零散分享提炼为简洁的阅读总结 | 被引用：Final Summary, SKILL.md

## 目标

用户读完一本书后，往往有很多感受但说不出、说不准。他们的分享是大白话、是情绪反应、是词不达意的直觉。洞察提炼的任务是：**帮用户把话说准，而不是替用户说话。**

## 输入

用户在阅读过程中的所有分享：
- 每单元结束后主动输入的文字
- 对引导观察的回应
- 中途插入的想法
- 全书完成后的感想

## 提炼规则

### 核心原则

1. **不复述情节**：去掉所有具体情节、人物名、地名
2. **保留判断**：只留用户的观点、感悟、价值判断
3. **抽象化升维**：把具体情节判断变成可迁移的人生洞察
4. **语气是用户的**：不是 AI 的文学评论，是用户"本该这么说"的话

### 提炼公式

```
用户原话 → 去掉情节 → 去掉人物名 → 保留判断 → 压缩到 ≤20字
```

### 示例

| 用户原话 | 提炼 |
|----------|------|
| "特丽克西对亲人的眷恋竟然对她产生那么大的影响" | 眷恋是最安静的力量 |
| "莫妮卡被安排走了一条最为曲折宏伟的路" | 最曲折的路未必通向答案 |
| "凡事莫向外求" | 凡事莫向外求（已足够精华，直接保留） |
| "信仰可以分离血脉" | 信仰可以分离血脉，但裂痕早就埋在血里了 |
| "归属感不在土地上，在人身上" | 归属感不在土地上，在人身上。失去人，就失去一切 |
| "帕依提提是寻找每个人心中可望却不可即之物" | 帕依提提从来不是一座城市，它是你心中那个可望不可即之物 |
| "谁能与时间和解谁才能获得内心的宁静" | 谁能与时间和解，谁才能获得内心的宁静。但大多数人只是被时间磨空了 |

### 质量标准

- **每条 ≤20 字**（含标点，偶尔可到 25 字）
- **3-6 条**：多了就不是精华
- **可迁移**：脱离这本书仍然成立
- **有节奏感**：读起来像格言，不像论文摘要
- **允许留白**：不把话说满，让读者自己补完

## 输出格式

```markdown
## 我的阅读总结

- {洞察1}
- {洞察2}
- {洞察3}
- ...
```

## 交互

1. 先展示提炼结果
2. 用户可以：
   - 直接接受
   - 修改措辞
   - 删除某条
   - 补充新的洞察
3. 最终版本写入 reading-journal.md

## 进阶：跨书积累

当用户读完多本书后，可以横向对比不同书的阅读总结：
- 哪些洞察反复出现？→ 可能是用户的核心关切
- 哪些洞察互相矛盾？→ 可能是用户的认知变化
- 哪些洞察越来越精准？→ 说明用户的表达能力在进步

这是长期目标，当前版本只做单书提炼。
